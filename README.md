# COMP 541 PROJECT
A replication of the paper [Improved Semantic Representations From Tree-Structured Long Short-Term Memory Networks](https://arxiv.org/abs/1503.00075) by Kai Sheng Tai, Richard Socher, and Christopher Manning.

# Preparation and Requirements
Java 1.8 is required for Stanford Corenlp 
As an interface between Stanford Corenlp, Python NLTK package is used and it should be installed
Stanford Corenlp java files should be set as environmental variables
